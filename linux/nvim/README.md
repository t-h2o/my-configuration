# Tips and tricks

```ctrl + z``` will suspend vim / neovim and ```fg``` (foreground) will re-display vim / neovim.

```ctrl +ww``` will change screen

```gt``` change tab

```gT``` change tab

remove highlight with ```:noh``` (no highlight)
