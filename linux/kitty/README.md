# My shortcuts

| keys						| effect			|
| :-:						| :-:				|
| ```ctrl + d```			| exit				|
| ```ctrl + o```			| increase opacity	|
| ```ctrl + shift + o```	| dicrease opacity	|
| ```ctrl + t```			| new tab			|
| ```ctrl + shift + t```	| close tab			|
| ```ctrl + tab```			| switch tab front	|
| ```ctrl + shift + tab```	| switch tab back	|
