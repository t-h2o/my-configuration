# My shortcuts

| keys						| effect			|
| :-:						| :-:				|
| ```ctrl + d```			| exit				|
| ```ctrl + shift + r```	| resize window		|
| ```ctrl + enter```		| split screen		|
| ```ctrl + ]```			| move on right win.|
| ```ctrl + [```			| move on left win.	|
| ```ctrl + o```			| increase opacity	|
| ```ctrl + shift + o```	| decrease opacity	|
| ```ctrl + t```			| new tab			|
| ```ctrl + shift + t```	| close tab			|
